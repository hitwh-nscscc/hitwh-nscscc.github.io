WHO ARE WE?

全国大学生计算机系统能力培养大赛 NSCSCC 是我国系统能力最高水平全国性学科竞赛[1]，
以自主设计“一个通用CPU、一个操作系统核心、一个编译系统”为总体目标。

哈尔滨工业大学（威海）全国大学生计算机系统能力培养大赛小组 - HITwh NSCSCC Team 是面向全校所有计算机硬件、软件爱好者的开放团队， 汇聚着威海校区来自于每个专业的系统上层、底层、硬件技术大牛， 亦代表着全校，甚至于所有三个校区中当届顶尖的计算机硬件技术水平[2] HITwh NSCSCC Team 每年将会选拔两个小队（每队1-4人）作为校队参加比赛。

除此之外，还会不定期举办技术讲座，分享来自于比赛最前线的、超越教学大纲的计算机硬件、软件底层关键技术。

[1] NSCSCC 全称是 National Student Computer System Capability Challenge, "最高水平"为官方描述

[2] 威海校区于第三届大赛预赛结果排名全国第三， 总决赛结果排名仅内部传阅

WHAT DO WE DO?

以自主设计“一个通用CPU、 一个操作系统核心、一个编译系统”为总体目标

加入团队吧！你可以接触到...

CPU Core[3]

一本指令集标准，到底能够描述一个多高程度的性能怪兽？
超标量体系结构、分支预测，到底能够设计出多高水准的通用处理器？
多核核间通信协议、循环缓冲器，如何优化能使性能评测大幅提升？

加入团队，发挥你天马行空的想象和超越常人的智慧，在 MIPS/ARM 中定义属于你自己的通用处理器最优架构！

System on Chip

中央处理器作为整个系统结构的核心，缺一不可却又渺小不堪。
NAND FLASH, DDR3 Memory，到底怎样构成一套存储系统？
MAC, UART，到底怎样从网络、串口与处理器进行交互？
Bootloader, Linux Kernel，到底如何才能运行一套完整的操作系统？

加入团队，针对比赛设计的 MIPS 处理器核心定义一个史无前例的片上系统，让处理器核心发挥出前所未有的高生产力！

Peripherals

掌握机器人运动学，想创造写字机器人？
对 IoT 独有一番品味，想智能联结世界？
玩转并行计算，想搭建高性能处理器集群？
深入理解 AXI3/AXI4/AXI4-Lite 总线协议？
脑放专家，Hi-Fi 大牛，IIS/IIC 倒背如流，想定义一个私人 Hi-Fi 系统？

加入团队，利用最新 AMBA 5 总线协议定义最具创造性的外部设备接口，针对比赛设计丰富实用的外设接口， 替团队在决赛中夺得一等奖迈出无可缺少的一大步！

Operating System

深度掌握 Linux 驱动框架，是一个性能狂人？
随意玩转用户态、核心态，穿梭于内存空间？
嵌入式大牛，移植如飞，还在利用 QEMU 来发挥自己的无敌天赋？

加入团队，发挥你对操作系统的独到理解，定制一个引人注目的 Linux Kernel，编写性能超群的底层硬件驱动，移植绚丽夺目的 上层应用，让竞争对手们无可超越！

Compiling System

执着于追求程序运行性能，看不惯 GCC 的鸡肋？
沉迷于汇编指令，对程序运行原理深有兴趣？

加入团队，为比赛设计的处理器核创造一个自己的编译系统，从程序级并行针对性地优化各种 corner condition，充分利用 CPU 资源， 让设计的处理器成为性能怪兽！
